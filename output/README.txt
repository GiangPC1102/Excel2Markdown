============================================================
THÔNG TIN THƯ MỤC OUTPUT
============================================================

Thư mục này chứa các file Markdown đã được chuyển đổi từ file Excel
trong thư mục "input".

Các file Markdown trong thư mục này được tạo ra tự động bởi ứng dụng 
Excel2Markdown.

Bạn có thể:
- Mở các file .md bằng bất kỳ trình soạn thảo văn bản nào
- Sử dụng nội dung Markdown trong các ứng dụng hỗ trợ Markdown
  (như GitHub, Notion, các CMS, blog, v.v.)

============================================================
CHÚ Ý:
- Nếu muốn chuyển đổi lại, chỉ cần chạy lại ứng dụng 
  (các file cũ sẽ bị ghi đè)
- Kiểm tra kết quả cẩn thận trước khi sử dụng vào mục đích 
  quan trọng
============================================================
