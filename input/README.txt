============================================================
HƯỚNG DẪN CHUYỂN ĐỔI FILE EXCEL SANG MARKDOWN
============================================================

1. Đặt tất cả các file Excel cần chuyển đổi vào thư mục này (input)
   - Hỗ trợ các định dạng .xlsx và .xls

2. Double-click vào file "Convert_Excel_To_Markdown.command" (Mac) hoặc 
   "Convert_Excel_To_Markdown.bat" (Windows) ở thư mục gốc

3. Tất cả các file Excel sẽ được chuyển đổi sang định dạng Markdown
   và lưu vào thư mục "output"

4. Kiểm tra kết quả trong thư mục "output"

============================================================
LƯU Ý:
- Ứng dụng sẽ giữ được định dạng của Excel như: merged cells, 
  bold, italic, căn chỉnh cột, v.v.
- Nếu gặp lỗi, vui lòng kiểm tra file Excel nguồn có lỗi hay không
